Links locais, externos e download de arquivos

Os links ou hyperlink são ligações que existem entre páginas. Vamos ver alguns exemplos do uso das âncoras.

Os links locais apontam para documentos que estão no seu próprio site. Acesse outra página nesse mesmo servidor clicando no link anterior.

Também podemos colocar documentos em outras pastas locais.Acesse outra página em outra pasta clicando no link anterior.

Também é possível acessar outros sites usando âncoras. Por exemplo, você pode criar um link para acessar o Youtube (não se esqueça de clicar no botão "VOLTAR" do seu navegador depois, para voltar para cá)

O link acima faz o usuário ir para outro site e abandona o meu. O ideal é que ele abra outros sites em uma nova aba. Acesse agora o Youtube em uma nova aba em branco.

Os links também podem ser usado para fazer o download de arquivos de outros formatos como ZIP, PDF, RAR, etc. Por exemplo, abra aqui um livro PDF de exemplo ou baixe a versão compactada do livro.

OBS:Clique sempre em exibir o código fonte para ver como tudo está sendo criado.
